# P+ Portfolio Plus｜AI 亮點捕捉器

> **P+：用 AI 演算你的核心競爭力。**

## 📖 專案簡介

**P+ Portfolio Plus** 是一款專為高中生設計的 AI 輔助工具，旨在優化學習歷程檔案中的「百字簡述」。

在大學申請過程中，教授往往只有極短的時間審閱檔案。本應用透過 Google Gemini AI 的強大語言模型，結合「影響力三角」與「四層次反思理論」，協助學生將雜亂的活動紀錄，提煉成高資訊密度、具備強大吸引力的黃金百字，讓教授在 3 秒內產生好奇心並點開附件。

## 🚀 核心功能與流程

本應用分為四個循序漸進的階段 (P+ Process)：

### 1. P+ Builder (建構)
*   **視覺化概念引導**：透過動態的「影響力三角模型 (Impact Triangle)」，引導學生理解動機 (Motivation)、行動 (Action) 與反思 (Reflection) 的黃金比例。
*   **目標設定**：確立「3 秒原則」——讓百字簡述成為一份精彩的電梯簡報。

### 2. P+ Refine (精煉)
*   **結構化輸入**：提供四種針對不同活動類型的撰寫模組：
    *   **改良 STAR**：適用於競賽、幹部、志工。
    *   **見解優先**：適用於閱讀心得、小論文。
    *   **問題解決**：適用於實驗、程式開發、專題。
    *   **關鍵字叢集**：適用於高度專業科系申請。
*   **即時引導**：針對選擇的架構提供具體範例與寫作公式。

### 3. P+ Fit (契合)
*   **AI 深度診斷**：
    *   **反思層級評估 (L1-L4)**：從單純描述 (L1) 到價值觀轉化 (L4) 的視覺化評分。
    *   **三維度健檢**：針對「主題明確度」、「資訊密度」與「冗詞贅字」進行自動分析。
*   **自動優化**：AI 生成一份符合 100 字限制、語氣專業且保留學生原意的優化版本。

### 4. P+ Insight (洞察)
*   **教授視角模擬**：輸入目標申請科系（如：臺大資工、政大傳播），AI 將切換至該領域教授的思維模式。
*   **Curiosity Score (好奇心分數)**：透過視覺化的儀表板，預測這份簡述吸引教授點開附件的機率。
*   **校系特化建議**：提供針對特定學群（理工/人文/商管）的關鍵字與寫作建議。

## 🛠️ 技術棧 (Tech Stack)

本專案採用現代化的前端技術構建，強調極致的 UI/UX 體驗與流暢的動畫效果。

*   **Frontend Framework**: React 19 (TypeScript)
*   **Styling**: Tailwind CSS (Custom Dark Mode / Glassmorphism Design)
*   **AI Integration**: Google Gemini API (`gemini-3-flash-preview`)
*   **Visualization**: Recharts (Radial Bar Charts), Custom SVG Animations
*   **Build Tool**: Parcel / Vite (ES Modules)

## 🎨 設計風格

*   **Cyberpunk / Tech Noir**: 深色背景 (`slate-900`) 搭配霓虹漸層 (`brand-blue`, `accent-violet`)。
*   **Glassmorphism**: 大量的毛玻璃效果與光暈背景，營造未來科技感。
*   **Micro-interactions**: 流暢的頁面切換、按鈕回饋與數據圖表動畫。
